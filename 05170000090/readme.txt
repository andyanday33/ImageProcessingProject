# Inside this directory, you will find:
Uygulama.py: Python code of a general purpose image processing application.
video_kenar_belirleme_opencv.py: A program that detects edges of every single object in a video given. Video must be on the same directory with the project and video must be named "Video.mp4".

Bu dizinin içerisinde bulunan
Uygulama.py : Geliştirilen genel özellikli uygulamanın açık kaynak kodudur.
video_kenar_belirleme_opencv.py : Video üzerinde kenar belirleme işlemi yapılması için yazılmış
	açık kaynak kodudur. (Kodun bulunduğu dizinde video.mp4 adında bir dosya bulunması gerekmektedir.)
Geriye kalan kodlar proje dosyasına dahil edilmemiştir, kodların ekran görüntüleri raporda bulunmaktadır.
